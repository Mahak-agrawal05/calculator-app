# Calculator App

A simple and interactive calculator web application built to practice JavaScript functionality and frontend development concepts.

## Features
- Perform basic arithmetic operations
- Interactive button-based interface
- Responsive and user-friendly design
- Real-time calculation display

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
This project was created to improve my understanding of JavaScript logic, DOM manipulation, and interactive web application development.
